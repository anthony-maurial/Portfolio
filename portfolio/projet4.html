Création d'un site Web

Accueil Météo IA IHM JPO

Contexte

Projet pédagogique en BUT Informatique du premier semestre. Réalisation d’un site web personnel dans le cadre de la SAE "Découverte du Web", afin d’apprendre les bases du développement web.

Objectifs

Concevoir un site statique complet sur le thème du bien être (j'ai choisi la musculation), en utilisant uniquement HTML, CSS et JavaScript. Structurer le site en plusieurs sections ou pages : présentation, exercices, conseils, etc.

Technologies

HTML pour la structure du contenu, CSS pour la mise en page responsive. Le tout développé sous VS Code avec Live Server pour l’aperçu en direct.

Ce que j’ai appris

Structuration HTML sémantique, stylisation CSS avec animations et adaptabilité mobile. J’ai aussi appris à gérer mon temps, à rechercher des solutions par moi-même, et à livrer un projet complet en autonomie.

Difficultés rencontrées

J’ai rencontré des problèmes de mise en page complexes résolus avec Flexbox, ainsi que des ajustements pour assurer la responsivité.
